# IT_Academy_Django

## Предварительные требования
Активация виртуального окружения

```bash
python -m venv .venv
.\.venv\Scripts\activate
```

Установка зависимостей Python

```bash
pip install -r requirements.txt
```

Миграции

```bash
python manage.py migrate
```

Запуск сервера

```bash
python manage.py runserver
```
